Natural Japanese. Command-line Freedom.

Unchiku is a japanese programming language. It is very useful especially for kids that don't know English. Also, it has support for general Kansai dialects.

⚠️　Press the ▶️ Play button to begin. There are four sample programs, number from "Sample 1.unchk", "Sample 2.unchk"...

  1. Press the ▶️ Play Button
  2. After ? File enter Sample 2.unchk.
  3. Sample 2.unchkis a program that calculated the total price of apples. If you input the price for each apple and the number of apples, the program will output the total price (subtotal).

    4. 誰でも簡単にテキストベースのプログラミングを楽しんでほしい、その思って「うんちく」を作りました。下のスライドと動画は、v0.3.0を「Tech Kids Grand Prix」の決勝プレゼンテーション（2018年9月）で発表した際のものです。
